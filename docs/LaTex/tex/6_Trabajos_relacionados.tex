\section{TFG relacionados}
\subsection{Retrieval-Augmented Generation para la Extracción de Información de Documentos Inteligente}
En este TFG se desarrolla un sistema RAG orientado al ámbito sanitario con el propósito de optimizar la recuperación y generación de información médica. El trabajo plantea una arquitectura basada en la recuperación y generación, que indexa documentación médica, crea \textit{embeddings} y orquesta \textit{prompts} para aportar contexto fiable al modelo generativo. Emplea técnicas de \textit{chunking} con solapamiento y un enfoque \textit{MultiQuery Retriever} para mejorar la cobertura y la precisión de la recuperación previa a la síntesis con un LLM. La solución se implementa sobre la API de \textit{Azure OpenAI}, utilizando \textit{LangChain} y \textit{FAISS} como motor de búsqueda vectorial, e integra una interfaz web en \textit{Flask} para simular consultas clínicas reales. Para la evaluación del sistema se emplea el \textit{framework RAGAS}, valorando métricas de precisión, relevancia y coherencia de las respuestas generadas, demostrando mejoras significativas frente a enfoques puramente generativos.~\cite{trabajorelacionado:Daniel}.
